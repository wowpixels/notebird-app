The NoteBird App is a note-taking app similar to Notion which uses Markdown language. The app is developed for educational purposes.

**Live:** [NoteBird App](https://notebird.wowpixels.app)

## Quick start

### Install dependencies

```bash
npm install
```

### Run the app

```bash
npm run dev
```
